Microwave_Oven_Project

1. Menu Screen
	SW1 - Micro mode
	SW2 - Grill mode
	SW3 - Convection mode
	SW4 - Start mode

2. Set Time function
	key - set Minute and Second use Matrix key pad
	* - Reset the current blinking field
	# - Enter to operation mode

3. Operation mode
	SW4 - Start / Resume & (Add 30 seconds)
	SW5 - Pause 
	SW6 - Stop (Terminate the process).

4. Convection mode
	key - Set temperature Maximum upto 180C and Minimum temperature > 0c.
	* - Reset the temperature
	# - Enter to Pre-Heating mode

5. Start mode
	SW4 - Start/Resume & (Add 30 seconds)
	SW5 - Pause
	SW6 - Stop the process
